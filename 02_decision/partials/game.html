Basic decision making

Basic layout: the world is unknown, better to say unexplored.

While the player explores the world (map is uncovering) he finds power ups.
If such power up is encountered, player chooses to pick it up and consume it. Exploration is done randomly - player keeps track of tiles he has not been to and if he has no other task to do, he picks up one tile and finds his way to it.

  • Health bonus - instant +20 of players health.
  • Speed bonus - +1 action per frame step, 3 frame steps timeout.

Decision tree status: For each step, the decision tree is traversed to find first node with non empty action and this action is the final decision for the moment.

Currently implemented actions:

  • If can see and object
    • If is object consumable - consume
    • Else explore the map further
  • Otherwise explore

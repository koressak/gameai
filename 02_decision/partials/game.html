Decision trees in unknown environment

Game goal

The main goal is to eliminate the enemy.
Each player starts with unexplored map, thus not knowing anything about the world around them.
As they explore the world searching for the enemy, they may find several bonus spawn points, which will help them achieve the goal.

Each player is able to fight with the enemy with his fists, inflicting damage (5 health points). The attack is possible while standing on the tile next to the enemy.

Bonuses

Bonuses help the player in these ways:

  • Health bonus - instant +20 of players health.
  • Speed bonus - +1 action per frame step, 5 frame steps timeout.
  • Damage bonus - +5 damage points.
Coding challenge

Main goal is to implement the Decision tree to simulate behavior of the game player.

Decision tree status

For each step, the decision tree is traversed to find first node with non empty action and this action is the final decision for the moment.

Currently implemented actions:

  • If can see and object
    • If is object consumable - consume
    • Else explore the map further
  • Otherwise explore

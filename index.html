Game AI by trial and error.

This page is for my personal project to get to know game artificial intelligence and corresponding algorithms.

The project evolution will go from basic pathfinding algorithms (as A*) to behavior trees for makeing decisions and, if God helps, to the neural networks or genetic algorithm to start the machine learning.

01 pathfinding

Basic A* pathfinding algorithm used to get player to the target - treasure chest.

Goal

Goal to this part is to implement A* algorithm. The player picks up random target (treasure chest) and finds it's way to it. When the chest is picked up by him/other player, he must seek other target to to pursue.

There are up to 5 players spawned when there is a chest consumed. Chests are spawned with 20% probability each game cycle.

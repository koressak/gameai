Game AI trial and error.

This page is for my personal project to get to know game artificial intelligence and corresponding algorithms.

The project evolution will go from basic pathfinding algorithms (as A*) to behavior decisions and, if God helps, to the neural networks or genetic algorithm to kepp the ai learning.

Goal

Goal to all parts of the project is a Player (AI) and a target. In pathfinding it's a player to find treasure chest. In others it will be PvP battle between players.

01 pathfinding

Basic Astar pathfinding algorithm used to get player to the target - treasure chest.

02 decision

Basic decision tree - building a tree of decisions and actions for player to interact in unknown world.
